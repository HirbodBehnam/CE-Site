Cloak Installer

If you want a proxy that works on any app, you can try the combination of shadowsocks + cloak; Cloak is a pluggable transport that uses "fake tls" to hide your traffic from DPI and authorities.
The script I wrote will install cloak on your server and If you want, you can also install shadowsocks alongside it via script to route your traffic through it.

Secure Forwarder

Sometimes security is more important than anything; in this case, you can use this small application I write to encrypt and forward any TCP connection to a server.
It supports 3 encryption algorithms, 5 key exchange methods and 3 transport modes to transfer your data.
However these days, I consider this project abandoned but you can use it for testing purposes. Using this project in a production environment is not recommended.

File to URL Bot

Sometimes you may want to schedule file downloads of Telegram or you need to give a file hosted in Telegram to an app that processes plain URLs.
In this case, you can use this really small bot to stream a file hosted on Telegram on a URL. This streaming means that file won't be downloaded on your server and thus zero disk will be used from your server.
